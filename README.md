# Minecraft Color Highlighter

VS Code extension that highlights Minecraft § color/formatting codes and `.lang` keys.

**Features**
- Highlights § color and formatting codes inside `.lang` files.
- Highlights `.lang` keys on the left side of `key=value` pairs.

**Try It**
1. Open `examples/demo.lang` in VS Code.
2. Make sure the language mode is `Minecraft Lang`.
3. You should see colored § codes and highlighted keys.

**Build (dev)**
1. `npm install`
2. `npm run compile`
3. Press `F5` to launch the Extension Development Host.

# Minecraft Color Highlighter (Русский)

Расширение VS Code для подсветки Minecraft § цветовых/форматных кодов и ключей `.lang`.

**Возможности**
- Подсветка § цветовых и форматных кодов внутри `.lang` файлов.
- Подсветка ключей `.lang` в левой части строк `key=value`.

**Как попробовать**
1. Открой `examples/demo.lang` в VS Code.
2. Убедись, что язык установлен на `Minecraft Lang`.
3. Ты увидишь подсветку § кодов и ключей.

**Сборка (dev)**
1. `npm install`
2. `npm run compile`
3. Нажми `F5`, чтобы запустить Extension Development Host.
